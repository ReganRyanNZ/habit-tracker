# 100% vibe-coded habit tracker

I've tried many habit tracking apps, but my favorite was just a google sheet where I could add all the bits I wanted, and remove all the bits I didn't want.

I especially like to have 2-3 person groups for habits, and the "add friend" functions on apps always fell short.

This is a PWA where you can create your own habit sheet, and add friends simply by sending them a url (no login required).

All habits are daily. All days are either "done" or blank. Keeping it simple.

# Architecture

I have no idea. This was a fun ~3 day experiment to see how far I could go with vibe coding. I've seen "prism" and "react" thrown around, so probably that's in there. I strictly didn't allow myself to edit a single line of code at all.

The results... ~pretty good. Many times I had to try 3-4 repetitions of "it's still not working, try again". But it's pretty good how far I could get in the end.
